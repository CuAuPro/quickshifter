# Quickshifter
Quickshifter for motorcycle.

<!-- ABOUT THE PROJECT -->
## About The Project

Quickshifter for motorcycles based on measured force on gearbox. Implemented is pull-type quicksihfter but it allows, with small modifications, push-type also.
